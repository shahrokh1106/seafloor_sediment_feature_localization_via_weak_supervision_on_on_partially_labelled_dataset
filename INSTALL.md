# Installation

Use **Python 3.10+**.

1. **PyTorch** — Go to [https://pytorch.org/get-started/locally/](https://pytorch.org/get-started/locally/), pick your platform and CUDA (or CPU), copy the command they give you, and run it in your environment.

2. **Ultralytics**

   ```bash
   pip install ultralytics
   ```

3. **Remaining packages** — from this folder:

   ```bash
   pip install -r requirements.txt
   ```
